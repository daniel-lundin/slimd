# slimd

Terminal slides from markdown

![img](slimd.gif)
## Usage

On node > 8, run with `npx`:

```
> npx slimd [markdown-file]
```

Otherwise, install globally with:

```
> npm install -g slimd
```

## Keys

 - **Space**, **Right arrow** - Navigate to next slide
 - **Left arrow** - Navigate to previous slide
 - **q**, **CTRL-C**, **ESC** - Quit
